# Web Scraper Project

## Overview

A Python-based web scraper that extracts quotes, authors, and tags from a website.

## Features

* Pagination support (multiple pages)
* Tag extraction
* Error handling
* User-Agent headers
* CSV export

## Tech Stack

* Python
* Requests
* BeautifulSoup

## Output

Structured CSV file containing scraped data.

## How to Run

```bash
python scraper.py
```
